## Tree Attributes

| 参数        | 说明           | 类型    | 默认值 |
| ----------- | -------------- | ------- | ------ |
| searchShow  | 是否节点过滤   | boolean | false  |
| placeholder | 搜索框提示文案 | string  | ''     |
